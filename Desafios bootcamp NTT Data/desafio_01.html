Aula 01 - Ferramentas Utilizadas

Índice

Antes de darmos início a nossa jornada de aprendizado de HTML, devemos instalar em nossos computadores algumas importantes ferramentas que irão auxiliar nosso desenvolvimento prático ao longo do curso. Dirija-se à página Visual Studio Code, baixe a versão compatível com seu sistema operacional e instale o software.

O Visual Studio Code, também chamado de VSCode, possui diversas funcionalidades, dentre elas, ele nos permite escrever documentos HTML, o que será útil para aplicação prática do aprendizado. O programa possui uma ampla biblioteca de extensões e para este curso utilzaremos a extensão Live Server, disponível na aba "Extensões". A extensão lança um servidor de desenvolvimento local, acessível via navegador, que possui recurso de recarregamento automático da página a cada vez que salvamos uma alteração no nosso documento. Com essas ferramentas, estamos prontos para o início de nossos estudos.

Aula 02 - Inspetor de Elementos

Índice

O inspetor de elementos é uma ferramenta de desenvolvimento disponível em diversos navegadores, para acessá-lo basta abrir o navegador e pressionar a tecla F12 do teclado, ou utilizar o atalho ctrl + shift + i, ou também podemos acessá-lo clicando nos três pontos horizontais localizados no canto superior direito ao lado da barra de endereços do navegador, navegar até "Mais ferramentas" e em seguida clicar em "Ferramentas do desenvolvedor" (caso esteja utilizando o google chrome).

Esta ferramenta disponibiliza um console com diversas funcionalidades, dentre elas a de inspecionar elementos na página. Selecionando o inspetor de elementos e em seguida passando o mouse sobre um elemento da página, podemos ver em destaque o HTML correspondente bem como o CSS utilizado. Podemos realizar alterações pontuais nos elementos que não atualizam o código da página, mas nos permitem ver as alterações realizadas em tela, até que recarreguemos a página.

Aula 03 - Estrutura Básica do HTML

Índice

A estrutura básica de um documento HTML é composta de basicamente três partes:

  1. Tag HTML - Indica que tudo que está entre a tag de abertura e de fechamento é um documento HTML.
  2. Tag Head - Onde colocamos as informações que serão carregadas antes de serem entregues ao usuário, contém instruções e dados da página, também é a área onde damos título à nossa página.
  3. Tag Body - É o corpo do documento, onde desenvolvemos tudo que aparecerá na tela para o usuário.

Aula 04 - Falando sobre Tags

Índice

As tags podem ser utilziadas para formatarmos o texto de uma página. Existem diversas tags, com diversas finalidades, abaixo citaremos alguns exemplos:

  • Tag i - A tag i aplica o formato itálico para o texto colocado entre sua abertura e fechamento, como este que você está lendo agora.
  • Tag strong - A tag strong aplica o aspecto negrito ao texto, como observável nesta frase.
  • Tag del - A tag del indica texto deletado (aplicando um riscado sobre o texto, como nesse parênteses).

Podemos fazer uso de tags de forma aninhada, onde utilizamos uma tag dentro da outra, gerando mais de um efeito no texto ao mesmo tempo, como o texto deste parágrafo que encontra-se em negrito e itálico.

Para finalizarmos essa aula, vale destacar que algumas tags não possuem fechamento, tendo abertura e fechamento em uma única tag, como por exemplo a tag input, a tag img, dentre outras.

Aula 05 - Atributos das Tags

Índice

Atributos são propriedades que as tags possuem que as modificam. Existem atributos gerais, que são comuns a todas as tags e atributos específicos de cada tag.

Exemplos de alguns atributos comuns:

  • id - o atributo id serve para identifcarmos aquela tag, podendo a tag ser reconhecida no Javascript a partir desse identificador, sinalizando para o Javascript onde ele irá mexer.
  • style - o atributo style é utilizado para se aplicar comandos css diretamente ao elemento que o contenha.
  • class - outro atributo voltado para o css. O nome dado a class serve como indicador para que o css realize alterações onde encontrar esse atributo com o nome escolhido.

Para finalizar, vejamos agora alguns atributos específicos:

  • type - a tag input, que gera um campo na página, possui este atributo, o qual indica o tipo daquele campo, que será textual, se será para inserção apenas de valores numéricos, dentre outros, podendo até mesmo delimitar o cmapo como um seletor de cor.
  • src - a tag img, utilizada para acréscimo de imagens ao documento, possui o atributo src que indica a fonte de onde se busca a imagem, para que ela seja gerada na página, seja através de um link remoto ou mesmo de um arquivo localizado em nossa máquina.

Aula 06 - Textos

Índice

Falaremos de algumas (somente algumas mesmo, são inúmeras) tags específicas para texto.

  1. tags h - as tags h são utilizadas como títulos e subtítulos de nossa página, indo desde a tag h1, que gera texto de tamanho maior (normalmente título de maior destaque), até a tag h6, de tamanho menor, marcando subtítulos menos relevantes ou seções na página.
  2. tag p - a tag p serve para delimitar como um parágrafo todo texto colocado entre sua abertura e seu fechamento.
  3. tag blockquote - a tag blockquote serve para destacar, em forma de citação o texto que vier entre sua abertura e fechamento, apresentando uma formatação textual própria.
  4. tag u - esta tag formata o texto para sublinhado, como esta frase exemplifica.
  5. tag sup - esta tag coloca o texto acima da linha como obsevável nestas palavras.

Aula 07 - Listas Ordenadas e Não Ordenadas

Índice

Existem três tipos de listas em HTML, as listas ordenadas, as listas não ordenadas e as listas de definição.

As listas ordenadas servem para criarmos listas enumeradas, enquanto que as não ordenadas geram listas marcadas por símbolos, ao invés de ordem numérica. Para criarmos uma lista ordenada utilizamos a tag ol e para criarmos os itens da lista utilizamos a tag li aninhada entre a tag ol de abertura e fechamento. Para a lista não ordenada utilizamos a tag ul para abertura e fechamento e aninhada entre elas utilizamos a tag li para os itens da lista, abrindo e fechando a tag para cada item da lista. Ao longo das aulas aqui descritas, utilizamos por diversas vezes as listas, ordernadas e não ordenadas, elas podem ser observadas nas aulas anteriores.

O terceiro tipo são as listas de definição, que utlizamos para delimitar um glossário, para que possamos listar termos e seus significados, utilizamos a tag dl para abertura e fechamento, utilizamos a tag dt para nomear o termo que será explicado e por fim a tag dd para o significado do termo. Aula 08 - Links

Índice

Para navergamos na internet de uma página para outra, normalmente clicamos em links nas páginas, mas como esses links são gerados? Através da tag a em conjunto com o atributo href, ao passarmos o link do documento que desejamos acessar para o atributo href, passamos a ter um elemento clicável que nos direcionará para outra página. Nesta tag adicionamos um texto, onde efetivamente clicaremos para acessar a página do link referenciado no atributo href.

Um dos atributos possíves da tag a é o target, onde podemos utilziar o nome _blank, permitindo que a partir do nosso clique a página seja aberta em outra janela/guia. Podemos também utilizar a tag para navegarmos nosso próprio site, bastando utilizar no atributo href o nome de nosso outro arquivo.

Podemos por fim, utilizar a tag a para navegarmos em nossa própria página, criando referências nas tags com o atributo id e as referenciando no href no formato #nome-do-seu-id-de-referencia. Nesta página criada há diversas ancoragens para facilitar a navegação por cliques, levando a aulas específicas ou ao início com apenas um clique.
